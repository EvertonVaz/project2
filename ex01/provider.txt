1. O passo a passo para obter uma chave de api.
    acesso o site: https://aistudio.google.com/api-keys
    faça login com sua conta google.
    clique em "Criar chave de API"
    de um nome para a sua api key
    selecione um projeto ou crie um novo projeto
    clique em "Criar chave" para gerar a chave de API
